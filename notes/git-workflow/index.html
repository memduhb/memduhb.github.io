Git Workflow Tips

Git Workflow Tips Link to heading

Basic Workflow Link to heading 

# Check status
git status

# Add changes
git add .

# Commit with message
git commit -m "descriptive message"

# Push to remote
git push origin main

Branching Link to heading 

# Create and switch to new branch
git checkout -b feature-branch

# Switch to existing branch
git checkout branch-name

# List all branches
git branch -a

# Delete branch
git branch -d branch-name

Stashing Link to heading 

# Stash changes
git stash

# List stashes
git stash list

# Apply latest stash
git stash pop

# Apply specific stash
git stash apply stash@{n}

Merging Link to heading 

# Merge branch into current
git merge branch-name

# Rebase current branch
git rebase main

# Abort rebase
git rebase --abort

Useful Commands Link to heading 

# View commit history
git log --oneline

# View file history
git log --follow filename

# Reset to previous commit
git reset --hard HEAD~1

# Create patch
git format-patch -1 HEAD
