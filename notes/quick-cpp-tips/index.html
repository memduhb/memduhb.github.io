RAII and Smart Pointers Link to heading

Always use smart pointers instead of raw pointers when possible:

// Good
std::unique_ptr<MyClass> ptr = std::make_unique<MyClass>();

// Avoid
MyClass* ptr = new MyClass();

Move Semantics Link to heading

Use std::move() to transfer ownership:

std::vector<int> v1 = {1, 2, 3};
std::vector<int> v2 = std::move(v1); // v1 is now empty

Const Correctness Link to heading

Mark methods as const when they don’t modify the object:

class MyClass {
public:
    int getValue() const { return value; } // Good
    void setValue(int v) { value = v; }    // Not const
private:
    int value;
};

Range-based For Loops Link to heading

Prefer range-based for loops when possible:

// Good
for (const auto& item : container) {
    // use item
}

// Avoid
for (auto it = container.begin(); it != container.end(); ++it) {
    // use *it
}

Lambda Expressions Link to heading

Use lambdas for short, local functions:

auto sum = [](int a, int b) { return a + b; };
int result = sum(5, 3); // 8
